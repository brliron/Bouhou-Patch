********** English *********
Reads the filenames of the bmp files in the "characters" folder and,
with the help of the dialogs*.txt files in the current directory, generates
a chars_dump.txt file.

********** French **********
Lit les noms de fichiers des bmp dans le dossier "characters",
et à l'aide des fichiers dialogs*.txt dans le dossier courant, génère
un chars_dump.txt en conséquence.
