********** English *********
Read the files dialogs*_all_chars_v2.txt, and removes the entries in it with just a number.
The result is stored in the input file (this is a minor step, no need to make the file names
even bigger just for that).
Note: this program is a bit flaky, you need to run it a few times to remove all the numbers.
