********** English *********
Read the files dialogs*_all_chars_v2.txt, and keep only the part of the texts that is fully displayed.
For example, if the input file have CAKE then CAKE_L, the output will have only CAKE_L.
The result will be in dialogs*_all_chars_v2 - jp sentences.txt

The order of the texts may change with the define WRITE_COMMENT_NOW. See the comment in the source
files for more details.

********** French **********
Lit les fichiers dialogs*_all_chars_v2.txt, et ne garde que la partie complètement affichée d'un texte.
Par exemple, si l'entrée contient CAKE puis CAKE_L, la sortie ne contiendra que CAKE_L.
Le résultat est stocké dans dialogs*_all_chars_v2 - jp sentences.txt

L'ordre des texts peut être influencé par le define WRITE_COMMENT_NOW. Voir le commentaire dans les sources
pour plus de détails.
